## TSS++

On ~~7/10~~ ~~7/20~~ 7/21 8:00 am PST. A new student info system called Triton Student System (TSS) was released. This replaced the beloved webreg system that helped UCSD students enroll and plan their courses for years. TSS recieved many complaints, not only ue to the delay in launch but also for its lack luster UI design and slow responses.

TSS++ Offers an alternative way to browse through the schedule of classes, and plan your schedule (without TritonGPT). It also features a live prerequisite viewer to assist with course planning

### Repo Structure

```
tssplusplus/
├── scrapers/                  the scraping pipeline
│   ├── catalog_scraper/       public catalog scraper -> data/catalog/<CODE>.json
│   ├── tss_scraper/           live TSS section/meeting scraper -> data/offered/<term>.csv + MongoDB
│   └── mark_offered_courses.py  cross-references the two, sets offered_this_qtr in data/catalog/
├── data/                      scraper output -- checked in, regenerated by the scrapers
│   ├── catalog/<CODE>.json    per-department course/prereq data
│   └── offered/<term>.csv     per-term list of offered courses
├── backend/                   (planned) API server reading data/ + MongoDB
├── frontend/                  (planned) web app consuming the backend API
└── requirements.txt           
```

## Getting Started

To get a local copy up and running follow these steps.

### Prerequisites

* Python 3.11+ and pip
* A MongoDB instance (local or Atlas) -- only needed for `tss_scraper`'s section/meeting data

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
2. Install Python dependencies
   ```sh
   pip install -r requirements.txt
   ```
3. For `tss_scraper`, add `MONGODB_URI` (and optionally `MONGODB_DB`) to a `.env` file at the repo root, and set up your TSS login cookie -- see [`scrapers/tss_scraper/README.md`](scrapers/tss_scraper/README.md) for the full walkthrough
4. Run the scrapers -- see [`scrapers/catalog_scraper/README.md`](scrapers/catalog_scraper/README.md) and [`scrapers/tss_scraper/README.md`](scrapers/tss_scraper/README.md)
5. More Steps Soon!
